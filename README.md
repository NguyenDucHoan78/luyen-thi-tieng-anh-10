# Luyen Thi Tieng Anh Lop 10

Website on tap mon Tieng Anh vao lop 10, gom 4 trang HTML tinh:

- `index.html`: trang chu tong hop
- `tu-vung.html`: hoc va on tap tu vung
- `chuyen-de.html`: luyen 21 chuyen de ngu phap
- `de-thi.html`: luyen 50 de thi thu

## Cach mo nhanh

Mo file `index.html` trong trinh duyet de vao trang chu va di chuyen den cac phan con.

## Noi dung hien co

- Giao dien ho tro che do sang / toi
- Tu vung theo unit
- Chuyen de ngu phap co giai thich va goi y on tap
- De thi thu co cham diem va luu tien do tren trinh duyet

## Cach cap nhat

Neu cap nhat du lieu hoac bo sinh website o thu muc du an goc, hay build lai de sinh ra 4 file HTML moi trong repo nay, sau do commit va push lai repo `website`.

## Luu y

Repo GitHub nay dang duoc dat truc tiep trong thu muc `website`, vi vay chi can push cac file HTML va `README.md` la du de cap nhat phien ban public.
